Natural C - Table of Contents

This is the home of the Natural C programming language, a dialect of C that adds a number of useful features to C.

Documentation

The manual is available online in several different formats:

  • The entire manual on a single HTML web page.
  • A multi-page HTML document with one page per section.

The API reference for the Natural C library is also available.

Download

Official releases can be downloaded from here.

To download a working copy of the Subversion repository, issue the following command: 

svn checkout http://naturalc.googlecode.com/svn/trunk/ ncc

The following sequence of commands will download, build, and install a copy of Natural C: 

svn checkout http://naturalc.googlecode.com/svn/trunk/ ncc
cd ncc
autoreconf -f -i
./configure
make
make install
